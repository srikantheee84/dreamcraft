**Tagline Options:**
1.  "Building Dreams. Crafting Reality."
2.  "Precision in Every Project."
3.  "Your Vision, Our Craft."
4.  "Dreamcraft: Built to Last."

## 2. Visual Identity System

**Color Palette (Light Mode):**

*   **Primary Blue:** `#1e3a8a` (Deep Royal Blue) - Used for Headings, Primary Buttons, Footer Backgrounds.
*   **Accent Yellow:** `#facc15` (Vivid Amber) - Used for CTAs, Highlights, Icons, Hover States.
*   **Neutral Light:** `#f8fafc` (Slate 50) - Main Page Background.
*   **Neutral White:** `#ffffff` (White) - Content Cards, Navbar.
*   **Text Dark:** `#334155` (Slate 700) - Body Text.

**Typography:**
*   **Headings:** *Montserrat* (Weights: 700 Bold, 600 SemiBold). Clean, geometric, structural.
*   **Body:** *Lato* (Weights: 400 Regular). Highly legible, friendly.

**Logo Usage Rules:**
*   **Placement:** Top Left in navigation.
*   **Clear Space:** Maintain at least 24px padding on all sides.
*   **Backgrounds:** Place strictly on White or Neutral Light gray. Do NOT place on dark colors (unless using a reversed white version).
*   **Restrictions:** Do not stretch the aspect ratio. Do not place directly on the Accent Yellow background (insufficient contrast).

## 3. Deployment Instructions

**Prerequisites:**
*   Node.js installed.

**Install & Run:**
```bash
npm install
npm start
```

**Deploy to GitHub Pages:**
1.  Update `package.json` with `"homepage": "https://<your-username>.github.io/<repo-name>"`
2.  Run `npm run build`
3.  Deploy the `dist` or `build` folder using `gh-pages`.

## 4. Flyer Template Strategy (Print)

**Layout:**
*   **Header:** White background. Logo Top Left. Phone number Top Right in Primary Blue.
*   **Hero Image:** Top 1/3rd. High-quality photo of a finished kitchen or exterior. Overlay text: "Transforming Spaces."
*   **Body:** Light Gray background section. Bulleted list of services (Kitchens, Baths, Additions).
*   **Offer:** Yellow band across the width. Text in Dark Blue: "Free Consultation & Estimate."
*   **Footer:** Deep Blue background. White text for address, license #, and website URL.

**Margins:** 0.5 inch safety margin. 0.125 inch bleed if printing professionally.
